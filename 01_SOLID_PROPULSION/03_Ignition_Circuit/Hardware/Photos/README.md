# Hardware Photos

## Overview

This folder contains reference photos for the Nakuja Project ignition circuit hardware.

Photos should help team members understand the physical assembly, wiring, PCB layout, connector arrangement, test setup, and hardware changes between revisions.

## Recommended Contents

```text
Photos/
├── README.md
├── Igniter_Board/
├── Trigger_Board/
├── PCB_Fabrication/
├── Sensor_Connections/
├── Power_System/
└── Test_Setup/
```

## Photos to Include

| Photo Type                 | Purpose                                                        |
| -------------------------- | -------------------------------------------------------------- |
| Igniter board top view     | Shows component placement                                      |
| Igniter board wiring       | Shows sensor and power connections                             |
| Trigger board top view     | Shows operator-side hardware                                   |
| Trigger board wiring       | Shows command and feedback connections                         |
| PCB fabrication photos     | Shows etching, drilling, soldering, or manufacturing process   |
| HX711/load-cell connection | Shows thrust sensor wiring                                     |
| DS18B20 connection         | Shows temperature sensor wiring                                |
| Pressure interface setup   | Shows pressure sensor/transmitter connection                   |
| Power system photo         | Shows battery, buck converter, boost converter, and connectors |
| Test setup photo           | Shows safe hardware arrangement during dry run or static test  |

## Documentation Notes

For important photos, add a short note explaining:

* What the photo shows.
* Which board revision it belongs to.
* Whether the setup was tested.
* Any known issue visible in the photo.
* Date taken, if available.

## Quality Guidelines

* Upload clear and well-lit photos.
* Avoid duplicate photos.
* Avoid blurry wiring photos.
* Compress very large images before uploading.
* Do not upload unrelated personal photos or unsafe test images without context.
