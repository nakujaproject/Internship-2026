# Dashboard Screenshots

## Overview

This folder contains screenshots of the Nakuja ignition web dashboard.

Screenshots are useful for documentation, progress reports, presentations, troubleshooting, and comparison between dashboard versions.

## Recommended Screenshots

| Screenshot | Purpose |
|---|---|
| Dashboard home screen | Shows the full dashboard interface |
| Connected dashboard | Shows successful telemetry connection |
| Recording mode | Shows data recording in progress |
| Event log example | Shows event tracking |
| CSV download example | Shows data export workflow |
| Mobile or resized view | Shows responsive layout if tested |

## Recommended Naming Format

Use clear file names:

```text
dashboard_home_v1.png
dashboard_connected_v1.png
dashboard_recording_v1.png
dashboard_event_log_v1.png
